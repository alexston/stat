查询语法

使用全文检索语法进行查询，最简单的检索条件是一个词，比如 hostname1、appserver。也可以使用通配符，比如查询所有的节点：

*

用下面的通配符，可以返回所有以 hostname 开头的节点

hostname*

可以使用通配符查询 IP 地址段：

192.168.1.*

可以对某个属性查询，这样更加精确：

networks.address:192.168.1.165

也可以查询一个地址段，这样：

networks.address:[192.168.1.100 TO 192.168.1.150]

对数值属性可以查询数值范围，比如要查询 CPU 大于 8 核的节点：

os.cpus:>8

可以使用 AND、OR 关联组合多个查询条件，比如要查询内存大于 8G，并且地址在某个段内的节点，可以这样：

memory.total:>8000000000 AND networks.address:[192.168.1.100 TO 192.168.1.150]

可以用标签属性查询：

tags:appserver AND tags:hpc
